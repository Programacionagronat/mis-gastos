Conectar con Google Sheets (igual que Mantenimiento)
1Abrí script.google.com. Creá un proyecto nuevo y pegá el contenido de apps_script.js.
2Reemplazá REEMPLAZAR_CON_TU_ID_DE_SHEETS con el ID de tu Google Sheet (la parte larga de la URL entre /d/ y /edit).
3Clic en Implementar → Nueva implementación → Aplicación web. Ejecutar como: Yo. Acceso: Cualquier usuario. Copiá la URL.
4Pegá esa URL acá abajo y hacé clic en Guardar.
Bot de Telegram — carga desde el celular
1Abrí Telegram → @BotFather → /newbot → seguí los pasos → copiá el token.
2Mandá un mensaje a tu bot y abrí: https://api.telegram.org/botTU_TOKEN/getUpdates — el número en "id" dentro de "chat" es tu chat_id.
3Creá un repositorio privado en GitHub llamado mis-gastos-bot. Subí bot_gastos.py y dentro de .github/workflows/ subí bot.yml.
4En el repo: Settings → Secrets → Actions → New secret. Creá: TELEGRAM_TOKEN, TELEGRAM_CHAT_ID, APPS_SCRIPT_URL (la URL de arriba).
5Listo. Cada 5 minutos GitHub revisa Telegram y guarda los gastos en tu Sheets automáticamente.
